# Holoflow

**Holoflow** is a high-performance image processing library designed for real-time and offline analysis. It models computation as a tree structure where nodes represent computational tasks and edges define dependencies.

## Features
- **Graph-based computation model**: Tasks are nodes, and edges represent dependencies.
- **Optimized for GPU**: Supports CUDA streams for parallel execution.
- **Memory management**: Tasks declare memory requirements, and the model handles allocations efficiently.
