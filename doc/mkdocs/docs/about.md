# About Holoflow

Holoflow is an open-source libraries collection for building high-performance signal processing applications. It aims to reduce the complexity of developing such applications by providing a modular and flexible framework. It was created by [Jules Guillou](https://github.com/JulesGuillou) in the research laboratory [CNRS](https://www.cnrs.fr/en) lead by [Prof. Michael Altan](https://www.michaelaltan.com/).

The initial goal of Holoflow was to support the needs of **Digital Doppler Holography** applied to real-time of retinal blood flow imaging. However, its modular architecture makes it suitable for a wide range of signal processing applications.
